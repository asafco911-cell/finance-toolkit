# Exercises

Standalone practice files from the early chapters of the course. Kept for reference
and continuity — these are **not** part of the production system.

| File | Chapter | Topic |
|---|---|---|
| `hello_finance.py` | 1 | First script — variables, types, f-strings |
| `file_practice.py` | 3 | File I/O — `with open()`, read/write modes |
| `json_practice.py` | 3 | JSON — `json.dump` / `json.load` |
| `storage.py` + `main.py` + `stocks.json` | 3 | Modules — importing across files |
| `first_api_call.py` | 4 | First LLM API call |

**Note:** these use relative paths and must be run from inside this directory.

The real projects live in the top-level directories — see the root `README.md`.
