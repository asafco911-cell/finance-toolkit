# Finance Toolkit

A Python finance toolkit for AI & Automation learning.

## Purpose
This repository contains tools and scripts for analyzing financial data, built as part of an AI & Automation Developer learning course.

## Getting Started
- Run: `python hello_finance.py`

## Course Context
This is Chapter 1 of a full Hebrew-language course: mastering terminal, Git, GitHub, and then building toward RAG systems on 10-K financial documents.

## Getting Started
- Run: `python hello_finance.py`
- Run: `python stock_screener.py` — classifies stocks by P/E ratio (Cheap/Fair/Expensive)
